HTML

  • HTML

    HTML (HyperText Markup Language) - це мова розмітки, яка використовується для створення веб-сторінок. Вона описує структуру та вміст веб-сторінки, а також те, як вона повинна бути візуалізована браузером.

  • Теги

    Теги HTML - це будівельні блоки веб-сторінок. Вони використовуються для опису структури та вмісту веб-сторінки, а також те, як вона повинна бути візуалізована браузером. теги бувають структурні, та контентні.

  • Структура

    HTML-документ має структуру, яка складається з тегів. Семантика тегів описує сенс елементів веб-сторінки. Важливо використовувати теги правильно, щоб веб-сторінка була доступною та зрозумілою для людей і пошукових систем.
